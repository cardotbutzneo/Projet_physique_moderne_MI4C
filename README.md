## Lancement :

### Prérequis intial :

Pour télécharger les dépendances nous vous conseillons d'utiliser un fichier venv.  
Pour créer le fichier tapez la commande :

```python
python3 -m venv .venv
```

Et pour l'exécuter :

```python
source venv/bin/activate
```

Pour lancer le programme, il suffit de taper la commande suivante :

```python
python3 resol_Schrodinger.py
```
Pour afficher les graphiques, rajoutez True à la fin de la commande précédente.
